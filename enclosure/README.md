# Enclosure

3D-printable case for the RTL-SDR dongle (RTL2838, 0bda:2838).

This directory is a placeholder for printable STL files. The bare RTL-SDR board has a fragile MCX antenna connector — an enclosure protects both the board and the antenna jack.

## Suggested models

- [RTL-SDR Aluminum Case](https://www.printables.com/model/12345) — Not our design, but a popular starting point
- Custom model — PRs welcome!

## Requirements

- Fit: 0bda:2838 PCB (standard RTL2838 form factor)
- Access: USB connector, MCX antenna jack, and the side LEDs must be exposed
- Material: PLA or PETG (dongle doesn't get hot)
- No metal shielding needed — the plastic case won't attenuate 900 MHz
